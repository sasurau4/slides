## bitとbyte

- bit
  - 情報量の単位
  - binary digitの略
  - 1bitで2進数の1桁で表せる情報量、つまり2つの状態を表せる
- byte
  - 情報量の単位
  - 1byte = 8bit = 256通りの値を示せる

## ArrayBufferとTypedArrayとDataView

<br>

### ArrayBuffer

- 固定長のバイナリデータを示すために使われる
- 直接操作ができない

<br>

### TypedArray

- ArrayBufferの配列状のビューを提供する
- Uint8Arrayなど

<br>

### DataView

- 任意のデータをバッファに読み書きする
- TypedArrayと違い、バイトオーダーを制御できる

---

## 実践

```js
const buf = new ArrayBuffer(1)
let uint8View = new Uint8Array(buf)

uint8View[0] = 65

String.fromCharCode.apply(0, uint8View)
// -> "A"
```
